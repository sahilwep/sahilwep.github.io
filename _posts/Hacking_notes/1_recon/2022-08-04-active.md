# Active Recon 

## **Ports Scanning**

### **nmap** 
* Check " nmap -h "
* Check man page
  

```bash
nmap -sS example.com -v # basic scan

nmap -sC -sV example.com -v # normal scan

nmap --min-rate=10000 -sC -sV example.com -v # fast scan

nmap --min-rate=10000 -p- example.com  -v   # Scan all ports
```
